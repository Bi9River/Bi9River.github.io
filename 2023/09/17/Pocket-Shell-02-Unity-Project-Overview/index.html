# Pocket Shell 02 Unity Project Overview

This post presents the Unity project overview of the Pocket Shell System, including the configuration, the project structure, and the project settings.

# Project Configuration

The project is configured to use the following settings:

  • Unity 2020.3.5f1c1 (c1 means it is a Chinese version)
  • .Net standard 2.0
  • Android 11.0
  • Windows 10

# Project Structure

The project structure is shown below:

  • Easy Character Movement
  • Joystick Pack
  • Mirror
  • Materials
  • Models
  • Pictures
  • Plugins
  • Prefabs
  • Resources
  • Scenes
  • Scripts
  • Textmesh Pro
  • Textures
  • Videos

# Project Settings

The project settings are shown below:

  • Rendering Pipeline: Universal Render Pipeline
  • Scripting Runtime Version: .Net standard 2.0
  • Scripting Backend: Mono
  • Input System: Old Input Manager
